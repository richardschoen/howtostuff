# Keys for tn5250
https://github.com/luca-vercelli/tn5250
```
Inside session, you can type:

^H = Left (backspace)
^R = Resume (if locked)
^P = Print
^? = Del
^M = Enter

^E, ^X = delete line?
^I = new line? (Horizontal Tab)
^J, ^K, ^L = delete to eol + new line? (Vertical Tab)

dixit: ^N/^O = toggle character set
```
